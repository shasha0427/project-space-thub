#  Project Space Thub

Project Space Thub is a full-stack MERN application built with a modular architecture that separates the Frontend and Backend. It includes modern UI components and secure backend APIs.

---

##  Features

- User authentication
- Admin dashboard
- Property listings
- Responsive design
- MongoDB integration
